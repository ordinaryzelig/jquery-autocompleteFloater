jQuery Autocomplete Floater

Project page: https://github.com/ordinaryzelig/jquery-autocompleteFloater

Description

Unobtrusively give select tags autocomplete functionality. In other words, leave the select tag alone!

Try it out

  • Click on the '+' to the right of the drop down.
  • Start typing an option to see jQuery autocomplete.
  • Click on an option.
  • The select tag will automatically change to that option.
